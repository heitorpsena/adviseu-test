# adviseu-test

## Instalação
-Clone o repositório:
</br>
`git clone https://github.com/heitorpsena/adviseu-test.git`
</br>
-Entre no diretório:
</br>
`cd adviseu-test`
</br>
-Instale as dependências:
</br>
`npm install`

## Rodando o Servidor
-Execute o seguinte comando:
</br>
<code> npm start </code>
